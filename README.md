# AI Certification Tracker · Road to CISO

An interactive, single-file study tracker for a 14-week AI / AI-security certification sprint on Coursera. Built to log daily study, track course completion, keep a weekday streak, follow a security-first plan, and stay visible on LinkedIn.

## Features
- **Dashboard** — weekday streak, hours this week, total hours, certs complete, today's focus.
- **14-Week Plan** — security-first sequence with the current week auto-highlighted.
- **Courses** — all six programs; tick courses as you finish; progress saved per device.
- **Daily Log** — record date / hours / course / notes; export & import a JSON backup.
- **LinkedIn** — milestone checklist + ready-to-use post templates.
- **Reminders** — opt-in browser notifications nudge you on weekdays if you haven't logged.

## Use it
Open `index.html` in any browser. Data is stored locally in your browser (`localStorage`) — use **Export backup** to save a copy. No server, no accounts, no tracking.

For a daily nudge even when the page is closed, a separate scheduled reminder runs in the Claude desktop app on weekday mornings.

## The six programs
1. Microsoft Copilot: Power Platform & Security Best Practices (Whizlabs)
2. IBM Generative AI for Cybersecurity Professionals
3. AWS Generative AI & AI Agents with Amazon Bedrock
4. Microsoft Generative AI Engineering
5. Google Cloud Generative AI Leader
6. Generative AI Strategic Leader (Vanderbilt)

See **STUDY-PLAN.md** for the full week-by-week breakdown.
